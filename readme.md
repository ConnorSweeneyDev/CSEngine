# CSEngine
A game engine for creating 2D/3D hybrid pixel art games with ease.

# How to Build
This project uses a custom 0 dependency bootstrapped build system called
[CSBuild](https://github.com/ConnorSweeneyDev/CSBuild).\
On windows, run `script\build.bat` in an environment with access to msvc development tools.\
On linux, run `script/build.sh` in an environment with access to gcc and g++.
